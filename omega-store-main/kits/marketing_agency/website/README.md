## How to Use

1. **Copy** this folder to `user-input/plug-and-play/kits/website/`
2. **Drop** existing work (wireframes, brand files) into `user-input/plug-and-play/`
3. **Run** the ignition prompt — AI will detect the kit
4. **Follow** the checklists in each phase
5. **Execute** task files as needed: "Run SEO.task.md"

---

## Task Files

| Task | Purpose |
|------|---------|
| `BRAND_IMPORT.task.md` | Import brand assets |
| `WIREFRAME_IMPORT.task.md` | Import wireframes/mockups |
| `CONTENT_AUDIT.task.md` | Audit existing content |
| `SEO.task.md` | Setup meta tags, sitemap, schema |
| `PERFORMANCE.task.md` | Optimize speed, images, caching |
| `ACCESSIBILITY.task.md` | WCAG compliance check |
| `LIGHTHOUSE.task.md` | Run Lighthouse audit |
| `MOBILE.task.md` | Test mobile responsiveness |
| `BROWSER.task.md` | Cross-browser testing |
