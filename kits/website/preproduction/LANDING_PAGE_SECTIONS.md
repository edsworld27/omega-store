# LANDING PAGE COMPONENT DICTIONARY

*This dictionary serves as the architectural foundation for selecting and collating landing page sections before coding begins.*

## 1. Hero Sections
- **Standard Hero:** Bold H1, Subheadline, Primary CTA, Secondary (Video) CTA.
- **Visual Split Hero:** Text on Left, large product image/animation on Right.
- **Cinematic Hero:** Full-screen background video or high-res image with centered, high-contrast overlay text.
- **Data-Driven Hero:** H1 with live metrics counter built-in.

## 2. Social Proof / Trust
- **Logo Cloud:** Horizontal scrolling ticker of trusted client/partner logos.
- **Metric Cards:** 3-4 Grid cards showing quantifiable success (e.g., "10k+ served", "99% uptime").
- **Featured Testimonial:** Singular, high-impact quote with customer portrait and credentials.

## 3. Product / Service Features
- **Zig-Zag Grid:** Alternating Image-Left/Text-Right layout down the page.
- **Bento Box:** Modern asymmetrical grid highlighting micro-features in distinct visual containers.
- **Icon Grid:** 3x2 grid of SVGs focusing on specific benefits with micro-copy.

## 4. Operational / Journey (How it Works)
- **Vertical Timeline:** Step-by-step numbered list leading down to a mid-page CTA.
- **Horizontal Process:** 3-Step horizontal flow (1. Audit -> 2. Build -> 3. Scale).

## 5. FAQs & Objections
- **Accordion FAQ:** Standard collapsing questions focused on removing friction (Pricing, Implementation length).

## 6. Closing CTA (The Anchor)
- **High-Contrast Footing:** Dark or vibrant block right before the footer.
- **Lead Capture Form:** Direct inline form (Email + Name -> Webhook).

---

**Usage Protocol:** During the `03_WIREFRAME_UX.task.md`, select a combination of these sections to form the architectural spine of the site before styling begins.
