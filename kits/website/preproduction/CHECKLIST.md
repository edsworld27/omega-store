# WEBSITE PRE-PRODUCTION CHECKLIST

Before you start building, make sure you have these ready.

---

## REQUIRED
- [ ] **Project definition** — What is this website for? Who is it for?
- [ ] **Sitemap** — List of pages and their hierarchy
- [ ] **Tech stack decision** — Framework, hosting, CMS (if any)

---

## RECOMMENDED
- [ ] **Brand guidelines** — Colors, fonts, logo, tone of voice
- [ ] **Wireframes or mockups** — Visual reference for layouts
- [ ] **Content inventory** — What copy/images exist or need creating
- [ ] **SEO keywords** — Target keywords for each page
- [ ] **Competitor analysis** — 2-3 reference sites

---

## IMPORT EXISTING WORK

If you've already built something (Lovable, v0, AI Studio, Figma), drop it in:

```
user-input/plug-and-play/
```

Then run the appropriate task file:
- `BRAND_IMPORT.task.md` — Import brand assets
- `WIREFRAME_IMPORT.task.md` — Import wireframes/mockups
- `CONTENT_AUDIT.task.md` — Audit existing content

---

## WHEN YOU'RE READY

1. All required items checked
2. Existing work imported (if any)
3. Run: "Execute CP-0 and CP-1"
